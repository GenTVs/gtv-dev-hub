# GTV Dev Hub

A GitHub-inspired developer hub for GTV Community projects.

## Project Goal

Build a simple developer platform to organize projects, experiments, bots, and learning progress.

## Phase 1 Features

- GitHub-style dark dashboard
- Navbar and sidebar layout
- Project cards
- Project detail page
- Search projects
- Create project modal
- Edit project
- Delete project
- LocalStorage persistence
- Hash routing
- Responsive mobile layout

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage
- Live Server

## Current Status

Phase 1 frontend prototype is complete.

## Next Phase

Phase 2 will focus on rebuilding the project with React and Vite.
